Cadastro de Viagens

Descrição:
Sistema para gerenciamento de viagens em obras, permitindo cadastrar, listar e controlar viagens, caminhões e obras.

Funcionalidades:
Cadastro de viagens, caminhões e obras
Listagem de viagens, caminhões e obras

Para Usar:
Basta baixar o release e executar a aplicação.

Tecnologias:
C#
.NET
Windows Forms
JSON
